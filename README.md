# CA Mailboxer 2018

Code base for week 2 challenge - Working with legacy code

## Installments
In order to run this program you will need to have the following installments:

* Rails 
> gem install rails
* Gems
> bundle install
* Rspec
> rspec-rails

Install your forked Repo to:
* Coveralls.io
* Semaphore
* Heroku
